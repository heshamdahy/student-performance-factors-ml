# Student Score Prediction ## 
📖 Project Overview This project focuses on **predicting students' exam scores** using Machine Learning techniques. 
The dataset used is 
**Student Performance Factors** from Kaggle. 
The workflow includes: 

- **Data Preprocessing:
- ** Cleaning the dataset, handling missing values, and outliers. 
- **Exploratory Data Analysis (EDA):** Understanding relationships between features and scores.
- - **Visualization:** Charts to explore key patterns in the data.
- - **Modeling: ** Training a **Linear Regression** model to estimate students' final scores based on study-related factors (e.g., study hours).
- - **Evaluation: ** Assessing model performance using regression metrics.
    
--- ## 🗂 Dataset The dataset includes features such as:
 -- Hours Studied
 -- Attendance
 -- Parental Involvement
 -- Access to Resources
 -- Previous Scores
 -- Motivation Level
 -- Internet Access
 -- Teacher Quality
  **Target Variable:** Exam_Score
  
--- ## 🛠 Tools & Libraries - Python - Pandas - Matplotlib - Scikit-learn
--- ## 📌 Topics Covered - Regression - Model Evaluation Metrics (MSE, MAE, R²)
